Kurulum (Geliştirici Modu)

Bu eklenti açık kaynaklıdır ve aşağıdaki adımları izleyerek tarayıcınıza doğrudan kurabilirsiniz:

Projeyi indirin:
Bu depoyu bilgisayarınıza klonlayın veya sağ üstteki "Code" butonuna tıklayıp "Download ZIP" seçeneği ile indirip bir klasöre çıkartın.

git clone https://github.com/[KullaniciAdiniz]/[RepoAdiniz].git


Chrome Eklentiler sayfasını açın:
Tarayıcınızın adres çubuğuna aşağıdaki adresi yapıştırın ve Enter'a basın:
chrome://extensions/

Geliştirici modunu aktif edin:
Açılan sayfanın sağ üst köşesindeki "Geliştirici modu" (Developer mode) anahtarını açık konuma getirin.

Eklentiyi yükleyin:
Sol üstte beliren "Paketlenmemiş öğe yükle" (Load unpacked) butonuna tıklayın.

Klasörü seçin:
Bilgisayarınıza indirdiğiniz ve zipten çıkardığınız eklenti klasörünü (içinde manifest.json dosyasının bulunduğu ana klasör) seçin.

Eklenti başarıyla yüklendiğinde tarayıcınızın eklentiler menüsünde görünür hale gelecektir.
